Renders an HTML radio type input to provide the ability for users to select on choice from a group of options.

```tsx
<Radio variant="radio" name="dark-mode" value="true" defaultChecked={true} />
```

## API

### Import

```tsx
import { Radio } from '@kodiak-ui/primitives'
```

### Props

| Name       | Type        | Default | Description                                                            |
| ---------- | ----------- | ------- | ---------------------------------------------------------------------- |
| variant    | string      |         | Name of the variant from the theme                                     |
| variantKey | string      | inputs  | Name of the variant key in the theme to know where to find the variant |
| sx         | SxStyleProp |         | Style object for the component                                         |
