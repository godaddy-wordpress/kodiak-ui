Image is used to quickly render the appropriate `img` tag and accept the `sx` and variant props.

```tsx
<Image src="https://api.adorable.io/avatars/400/abott@adorable.io.png" />
```

## API

### Import

```tsx
import { Image } from '@kodiak-ui/primitives'
```

### Props

| Name       | Type        | Default | Description                                                            |
| ---------- | ----------- | ------- | ---------------------------------------------------------------------- |
| variant    | string      |         | Name of the variant from the theme                                     |
| variantKey | string      |         | Name of the variant key in the theme to know where to find the variant |
| sx         | SxStyleProp |         | Style object for the component                                         |
