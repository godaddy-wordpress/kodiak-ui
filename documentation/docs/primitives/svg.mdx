Used to render a SVG icon. The `fill` attribute is automatically set to `currentColor` to allow
you to update the styling based on the `color` CSS property.

```tsx
<SvgIcon
  viewBox="0 0 24 24"
  title="Search icon"
  desc="Search magnifying glass icon"
  sx={{
    color: 'primary',
    mr: 2,
    height: '24px',
    width: '24px',
  }}
>
  <path d="M15.5 14h-.79l-.28-.27C15.41 12.59 16 11.11 16 9.5 16 5.91 13.09 3 9.5 3S3 5.91 3 9.5 5.91 16 9.5 16c1.61 0 3.09-.59 4.23-1.57l.27.28v.79l5 4.99L20.49 19l-4.99-5zm-6 0C7.01 14 5 11.99 5 9.5S7.01 5 9.5 5 14 7.01 14 9.5 11.99 14 9.5 14z" />
</SvgIcon>
```

## API

### Import

```tsx
import { SvgIcon } from '@kodiak-ui/primitives'
```

### Props

| Name       | Type        | Default | Description                                                            |
| ---------- | ----------- | ------- | ---------------------------------------------------------------------- |
| variant    | string      |         | Name of the variant from the theme                                     |
| variantKey | string      | inputs  | Name of the variant key in the theme to know where to find the variant |
| sx         | SxStyleProp |         | Style object for the component                                         |
